What is Streamlit?

  • Python library for turning scripts into apps
  • No front-end coding required
  • Great for dashboards & prototypes

Use cases: business dashboards, internal tools, quick prototypes

Demo: Hello Streamlit


pip install streamlit
streamlit hello

Your First App


import streamlit as st

st.title("My First Streamlit App")
st.write("Hello, world! This is a simple interactive web app.")

Run with:

streamlit run app.py

Widgets & Interactivity

  • st.button("Click me")
  • st.text_input("Enter your name")
  • st.slider("Pick a number", 0, 100)
  • st.selectbox("Choose option", [...])

Example: Widgets


st.title("Interactive Widgets")

name = st.text_input("Enter your name:")
age = st.slider("Select your age", 0, 100, 25)

if st.button("Submit"):
    st.write(f"Hello {name}, you are {age} years old!")

Displaying Data


import pandas as pd
import numpy as np

st.header("Displaying Data")

df = pd.DataFrame({
    "Products": ["A", "B", "C"],
    "Sales": [100, 200, 150]
})
st.table(df)

chart_data = pd.DataFrame(
    np.random.randn(20, 3),
    columns=["Product A", "Product B", "Product C"]
)
st.line_chart(chart_data)

File Upload


uploaded = st.file_uploader("Upload a CSV file", type=["csv"])
if uploaded:
    data = pd.read_csv(uploaded)
    st.write("Preview of Data", data.head())

Filtering Example


column = st.selectbox("Choose a column", data.columns)
st.write(data[column])

Layouts & Organization

  • Sidebar: st.sidebar.widget()
  • Columns: col1, col2 = st.columns(2)
  • Tabs: tab1, tab2 = st.tabs([...])

Example: Sidebar & Columns


st.sidebar.header("Sidebar Filters")
option = st.sidebar.selectbox("Choose an option", ["Overview", "Details"])

col1, col2 = st.columns(2)
col1.metric("Sales", "200K")
col2.metric("Profit", "50K")
